# Adrian's Dotfiles

## Install

```sh
git clone https://github.com/TheCheeseTown/dotfiles.git && cd dotfiles && source install.sh
```

Or without cloning first:

```sh
bash -c "$(curl -fsSL https://raw.githubusercontent.com/TheCheeseTown/dotfiles/main/install.sh)"
```

## Update dotfiles

```sh
source bootstrap.sh
```

## Homebrew

```sh
/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"
```
